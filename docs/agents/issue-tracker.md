# Issue tracker: Linear

Issues and PRDs for this repo live in Linear.

Workspace: `threadfork`
Team: `GDM`
URL: https://linear.app/threadfork/team/GDM/all

Use the Linear connector/app when available. When a skill says "publish to the issue tracker", create a Linear issue in the `GDM` team. When a skill says "fetch the relevant ticket", read the referenced Linear issue from that team.

Use the triage vocabulary in `docs/agents/triage-labels.md` for labels or equivalent Linear workflow metadata.
