## Agent skills

### Issue tracker

Issues are tracked in Linear, in the `threadfork` workspace's `GDM` team. See `docs/agents/issue-tracker.md`.

### Triage labels

The default triage vocabulary is used: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

This repo uses a single-context domain-doc layout. See `docs/agents/domain.md`.
